Eye Examination OSCE Checklist

Teaching-adapted structure referencing Geeky Medics eye examination OSCE resources. Use local trust pathways and supervision requirements in clinical settings.

Core sequence (general)

  1. Hand hygiene; introduce self/role; confirm identity; explain exam; gain consent; check pain.
  2. Distance visual acuity: each eye separately with correction; record; pinhole if reduced; CF/HM/PL if needed.
  3. Near visual acuity (if relevant); consider colour vision if indicated.
  4. Visual fields to confrontation (screen).
  5. External inspection: lids, conjunctiva, cornea clarity; pupil size/shape.
  6. Pupils: direct/consensual responses; assess for RAPD if indicated.
  7. Ocular movements; ask about diplopia or pain on movement.
  8. Red reflex and anterior segment assessment (if equipment available).
  9. Fundoscopy if view possible; otherwise document limitation and plan.
  10. Summarise findings; safety-net; propose next steps/referral pathway as appropriate.

Safety add-on (glaucoma red flags)

  1. Ask specifically: acute pain, headache, nausea/vomiting, halos/blur, rapid onset, unilateral symptoms.
  2. Look for: red eye, corneal haze, mid-dilated poorly reactive pupil (if present, treat as urgent).
  3. If acute angle-closure suspected: escalate urgently and follow local protocol / ophthalmology advice.
